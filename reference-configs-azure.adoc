== 授權支援的組態

Cloud Volumes ONTAP 在 Azure 中既可以作為單節點系統使用，也可以作為高可用性（HA）節點對使用，以實現容錯和不中斷營運。

不支援將單節點系統升級為 HA 配對。如果您想要在單節點系統和 HA 配對之間切換，則需要部署新系統，並將資料從現有系統複寫到新系統。

Cloud Volumes ONTAP 可以在雲端供應商提供的預留或隨需 VM 執行個體上執行。不支援使用其他 VM 執行個體類型的解決方案。

有關支援的執行個體規格，請參閱  https://learn.microsoft.com/en-us/azure/virtual-machines/sizes/overview["Microsoft Azure 說明文件"^]。



=== 單節點系統
